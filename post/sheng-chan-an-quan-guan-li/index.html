安全

安全是指在生产活动中，能将人或物的损失控制在可接受水平的状态。

危险

危险是指在生产活动过程中，人或物遭受损失的可能性超过了可接受范围的 一种状态。

风险（危险性）

风险是描述系统危险程度的客观量。可以表示为 R=f(p,c) 。其中 c 表示损失程度， p 表示发生概率， R 表示风险。总风险表示为 总风险=f( 人的因素 + 设备因素 + 环境因素 + 管理因素 + 其他因素 )

安全性

安全性为衡量系统安全程度的客观量。

事故

定义还不一致。可以分为

  1. 特别重大事故
  2. 重大事故
  3. 较大事故
  4. 一般事故

事故隐患

隐藏的祸患。

危险源

危险源是指可能造成人员伤害、疾病、财产损失、作业环境破坏或其他损失的根源或状态。

  1. 第一类危险源是指系统中存在的、可能发生意外释放的能量或危险物质
  2. 第二类危险源是指导致能量或危险物质约束或限制措施破坏或失效的各种因素。

重大危险源

重大危险源就是可能导致重大事故发生的危险源。

系统安全

系统安全是指在系统寿命期内应用系统安全工程和系统安全管理方法，辨识系统中的危险源，并采取控制措施使其危险性最小，从而使系统在规定的性能、时间和成本范围内达到最佳的安全程度。

可靠性和安全性都是判断、评价系统性能的重要指标。

本质安全

本质安全是指设备、设施或技术工艺含有内在的能够从根本上防止事故的功能。包括两方面

  1. 失误 - 安全功能
  2. 故障 - 安全功能

中国的安全生产方针

安全第一、预防为主、综合治理

安全管理制度

以安全生产责任制为核心。安全生产责任制、安全技术措施计划、安全生产教育、安全生产检查以及伤亡事故的调查和处理制度构成了我国的企业安全管理基本制度，称为 “ 五项制度 ” 。

安全生产责任制

按照 “ 安全第一、预防为主、综合治理 ” 安全生产方针和 “ 管生产必须管安全，谁主管谁负责 ” 的原则。主要的负责人有

  1. 企业的主要负责人
  2. 生产经营单位其他负责人
  3. 生产经营单位各职能部门负责人及其工作人员
  4. 班组长
  5. 岗位工人

安全分析方法的特点及适用范围

  1. 统计图表分析：定量分析，系统发生事故。
  2. 因果分析图：分层分析、用途广泛。
  3. 安全检查表：检查设计、系统和工艺过程。
  4. 预先危险性分析：检修后开车、制定操作规程、技术改造后、使用新工艺等。
  5. 故障模式及影响分析：以硬件为对象，主要用于设计阶段。
  6. 危险性和可操作性研究：适用于流体或能量的流动情况分析。
  7. 事件树分析：能够分析出各种事件发展的可能性结果，动态的宏观分析方法。
  8. 事故树分析：适用于找出各种失效事件之间的关系。

轨道交通运输的产品是乘客的位移。
通常把列车的组织和运行工作统称为行车工作。
员工的人身安全、设备安全和乘客人身安全。
行车安全包括：行车调度安全、列车驾驶安全、车站作业安全、接发列车作业安全、调车作业安全等。

车辆破损分为：

  • 报废
  • 大破
  • 中破
  • 小破

行车事故等级分为：

  • 载客列车重大事故
  • 载客列车大事故
  • 险性事故
  • 一般事故

行车事故的危害性。

  • 造成人民生命财产的损失与损害。
  • 造成国家和财产的严重损失，给企业的经济效益造成损失。
  • 给城市轨道交通运输的正常秩序造成紊乱，严重影响乘客出行。
  • 严重的行车事故将会给城市轨道交通的形象以及社会造成十分恶劣的负面影响。
